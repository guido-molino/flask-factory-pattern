# Python - Flask

A dockerized base environment to develop minimal web applications.

## Setup

1. `git clone`
2. `docker-compose up`
3. project gets served at: http://localhost:8007

## Versions

- Python 3.8.12
- [Pipenv](https://pipenv.pypa.io/en/latest/) (python package manager)
- Flask Framework 2.0.3
- MySQL 8.0
